# 🌍 AirVision – Air Quality Analysis Dashboard

AirVision is a responsive web application built using **HTML**, **CSS**, **JavaScript**, and **Chart.js** to visualize and analyze air quality data. The dashboard presents key AQI statistics, interactive charts, city-wise air quality information, search functionality, and data export options in a clean and modern interface.

---

## 🚀 Features

- 📊 Displays key air quality statistics
  - Average AQI
  - Highest AQI
  - Cleanest City
  - Most Polluted City
- 📈 Interactive City-wise AQI Bar Chart
- 🥧 Pollutant Distribution Doughnut Chart
- 🔍 Real-time city search
- 🔄 Refresh button with simulated live AQI updates
- 📋 Responsive data table
- 📤 Export air quality data as CSV
- 📤 Export air quality data as JSON
- 📱 Fully responsive design
- 🎨 Modern dark-themed user interface
